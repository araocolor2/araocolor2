# 제주컬러 (arao.kr) 상세 개발 계획서

> `@` 표시 항목은 인터뷰 보완 과정에서 추가된 내용입니다.

## 서비스 개요

| 항목 | 내용 |
|---|---|
| 브랜드명 | 제주컬러 |
| 도메인 | arao.kr |
| 서비스 목적 | 자연스럽고 간편한 디지털 상품 결제 경험 구축 |
| 주요 고객 | 30~50대 모바일 사용자 |
| 상품 종류 | 디지털 상품 (다운로드) |
| 상품 가격대 | 11만원 내외 |
| 구매 방식 | 1회성 구매 |
| 오픈 목표 | 1개월 이내 |

---

## 기술 스택

| 영역 | 선택 | 역할 |
|---|---|---|
| 프레임워크 | Next.js 15 | 전체 웹 앱 |
| @ 인증 | Supabase Auth | 구글 로그인 (1차), 카카오 (2차) |
| @ 이메일 발송 | Resend | 구매 완료 이메일 발송 (무료 월 3,000건) |
| 데이터베이스 | Supabase | 사용자, 게시글, 댓글, 구매 데이터 |
| 파일 저장 | Supabase Storage | 이미지 업로드, 디지털 상품 파일 |
| 알림 | Supabase Realtime | 실시간 알림 |
| 결제 | Stripe | 카드 결제, 쿠폰, 이메일 영수증 |
| 전화 인증 | Twilio Verify | 첫 구매 시 SMS 인증 |
| 배포 | Vercel | 호스팅 |
| 통계 | Google Analytics | 방문자 통계 |

## 사용자 등급 & 권한

| 등급 | 접근 범위 |
|---|---|
| 비로그인 | 글 목록 + 댓글 수 + 좋아요 수만 |
| 일반 회원 | 공용 게시판 전체 읽기/쓰기 |
| @ 구매 회원 | 공용 게시판 + 제주컬러 전용 게시판 (어떤 상품이든 1개 이상 구매 시 자동 등급 상승) |
| 관리자 | 전체 + 회원 관리 + 상품 관리 + 구매 내역 |

## 구매 & 결제

| 항목 | 내용 |
|---|---|
| 결제 수단 | 신용카드 (Stripe), 무통장입금은 사업자 등록 후 추가 |
| 결제 통화 | 원화 (KRW) |
| 결제창 | Stripe 기본 결제창 |
| @ 장바구니 | DB 저장 + 로컬스토리지 병행, 로그인 시 자동 병합 |
| @ 중복 구매 | **같은 상품 여러 번 구매 가능, 주문은 건별로 저장** |
| @ 구매 권한 계산 | **다운로드/구매회원 권한은 "유효한 주문" 기준으로 계산 (환불 완료 주문 제외)** |
| 쿠폰 | **쿠폰 코드 입력 (Stripe Coupon 활용), 만료 쿠폰 사용 불가, 1회 주문당 1개만 적용, 중복 적용 불가, 최소 결제금액 조건 미충족 시 적용 불가** |
| 전화 인증 | **회원 계정 기준 첫 구매 시 1회만 (Twilio SMS)** |
| 이용약관 동의 | 결제 전 체크박스 동의 |
| 환불 | 불가 (구매 버튼 아래 안내 텍스트 표시) |
| @ 결제 확정 기준 | **결제 완료 신호(Stripe 웹훅) 확인 후 주문 확정 처리** |
| 결제 완료 후 | **주문 확정 후** 마이페이지 구매 내역으로 이동 |
| 결제 실패 시 | **실패 로그 저장 +** 실패 메시지 + 재시도 버튼 |
| @ 결제 취소/실패/환불 구분 | **사용자 결제창 이탈·취소: 미결제로 종료 / 결제 실패: 재시도 유도 / 환불 완료: 구매 권한 회수** |

| 테이블 | 역할 |

| users | 회원 정보, 등급, 알림 설정, 전화번호, 탈퇴 여부 |

| comments | 댓글 + 대댓글 (parent_id 연결) |
| likes | 게시글/댓글 좋아요 |
| products | 상품 정보 (상품명, 가격, 설명, 파일, 미리보기) |
| orders | 구매 내역 (구매자, 상품, 금액, 결제 수단, 쿠폰) |
| download_logs | 다운로드 횟수 및 이력, 링크 만료일 |
| cart | 장바구니 (DB + 로컬스토리지 병행) |
| notifications | 알림 (수신자, 내용, 날짜, 읽음 여부) |
| inquiries | 문의 내역 (유형, 내용, 처리 상태) |
| coupons | 쿠폰 코드, 할인율, 사용 여부 |
| user_deletions | 탈퇴 이력, 탈퇴일, 복구 가능 여부 |
| reviews | 상품 후기 (별점 1~5, 텍스트, 구매자만 작성) |
