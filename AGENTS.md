
## 참조 문서인가? 

claude.ai 는 이 문서를 참조하지 말것.
gpt5.3 만 이문서에 책임을 가질것
CLAUDE.md 는 작업 기준 문서에서 제외하며, 읽기/인용/참조 금지.

## 규칙

어려운 기술용어는  최소한으로 사용 
페이지 파일 이름 표시금지
테이블 표를 활용, 중간 글 제목도 쓰고 

코등결과 보고할떄는 300줄 이내 
웹사용자 경험 측면에서 이해하기 쉽게 설명하기
현재 기준 Stripe 는 사용하지 않으며, 결제는 카카오 링크 결제와 카카오뱅크 무통장 입금 기준으로 진행
작업 결과를 제시한 뒤에는 자동 검증(예: lint/build/실행 흐름 확인)을 반드시 수행
chatgpt 5.3 이 프로젝트 기획에서 매우 잘못된 정보로 인해  프로젝트가 중지
