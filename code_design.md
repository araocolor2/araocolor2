# 제주컬러 디자인 컨셉

> 참조: `refer/design.md` (Apple 디자인 시스템)

---

## 전체 무드

| 항목 | 내용 |
|---|---|
| 키워드 | 제주 자연 + Apple 감성 — 깨끗하고 여백이 넉넉하며 신뢰감 있는 느낌 |
| 배경 리듬 | 섹션마다 흰색(`#ffffff`) ↔ 연한 회색(`#f5f5f7`) 교차 (Apple 방식) |
| 분위기 | 미니멀하되 차갑지 않게 — 제주 바다의 청록으로 온기 추가 |

---

## 색상

| 역할 | 색상 | 코드 |
|---|---|---|
| 강조 (버튼, 링크, 뱃지) | 제주 청록 | `#0a7ea4` |
| 어두운 배경 강조 링크 | 밝은 청록 | `#34c4e8` |
| 기본 텍스트 (밝은 배경) | 거의 검정 | `#1d1d1f` |
| 보조 텍스트 | 80% 검정 | `rgba(0,0,0,0.8)` |
| 약한 텍스트 (날짜, 카운트) | 48% 검정 | `rgba(0,0,0,0.48)` |
| 밝은 배경 | 연한 회색 | `#f5f5f7` |
| 카드 배경 | 흰색 | `#ffffff` |
| 어두운 섹션 배경 | 거의 검정 | `#000000` |
| 어두운 카드 배경 | 다크 서피스 | `#272729` |
| 흰색 텍스트 (어두운 배경) | 흰색 | `#ffffff` |
| 에러 | 빨강 | `#ff3b30` |
| 성공 | 초록 | `#34c759` |

> Apple Blue(`#0071e3`) 대신 제주 청록(`#0a7ea4`)으로 대체. 나머지 색상 체계는 동일하게 유지.

| 비활성 버튼 | `rgba(0,0,0,0.1)` | `rgba(0,0,0,0.3)` | 8px | 비활성 상태 |

## 카드 & 컨테이너

| 항목 | 내용 |
|---|---|
| 배경 | `#ffffff` (밝은 섹션) / `#272729` (어두운 섹션) |
| 테두리 | 없음 |
| 모서리 | 8px |
| 그림자 | `rgba(0,0,0,0.22) 3px 5px 30px 0px` |
| 호버 | 그림자 살짝 강해짐 |

## 네비게이션

| 항목 | 내용 |
|---|---|
| 상단 헤더 | 투명 다크 (`rgba(0,0,0,0.8)`) + blur 효과 (스크롤 시 고정) |
| 높이 | 48px |
| 로고 | 왼쪽 정렬 |
| 벨 아이콘 | 오른쪽 (읽지 않은 알림 청록 뱃지) |
| 하단 탭바 | 아이콘만 4개 (홈 / 게시판 / 상품 / 마이페이지), 웹에서 마우스오버 시 텍스트 |
| 탭바 높이 | 56px (안전 영역 포함) |
| 선택된 탭 | 청록 아이콘 (`#0a7ea4`) |

## 여백 & 간격

| 항목 | 값 |
|---|---|
| 기본 단위 | 8px |
| 화면 좌우 여백 | 16px (모바일) / 32px (데스크톱) |
| 섹션 간 여백 | 48px ~ 80px |
| 카드 내부 여백 | 16px ~ 24px |
| 버튼 패딩 | 8px 16px |

## 이미지 처리

| 항목 | 내용 |
|---|---|
| 상품 이미지 | 단색 배경(흰색 또는 `#f5f5f7`) 위에 상품만 — 배경 없음 |
| 홈 히어로 | 풀스크린 제주 바다 이미지, 텍스트 오버레이 |
| 게시글 썸네일 | 200×200 정사각형, 8px 모서리 |
| 비포/애프터 슬라이더 | 상품 상세 페이지 상단 |

---

## 모서리 반경 기준

| 크기 | 값 | 용도 |
|---|---|---|
| 소 | 5px | 태그, 뱃지 |
| 중 | 8px | 버튼, 카드, 이미지 |
| 대 | 12px | 모달, 바텀시트 |
| 알약 | 980px | 텍스트 링크 버튼 |
| 원형 | 50% | 프로필 사진, 미디어 버튼 |

---

## 깊이 & 그림자

| 단계 | 처리 | 용도 |
|---|---|---|
| 플랫 | 그림자 없음 | 일반 콘텐츠 영역 |
| 글래스 네비 | `rgba(0,0,0,0.8)` + blur | 상단 헤더 |
| 카드 | `rgba(0,0,0,0.22) 3px 5px 30px` | 상품 카드, 팝업 |
| 포커스 | `2px solid #0a7ea4` | 키보드 접근성 |

---

## 금지 사항

- 추가 강조 색상 사용 금지 — 청록 하나만
- 그라디언트, 텍스처, 패턴 배경 금지
- 카드에 테두리 금지
- 본문 텍스트 가운데 정렬 금지 (헤드라인만 가운데)
- 두꺼운 그림자 여러 겹 금지
