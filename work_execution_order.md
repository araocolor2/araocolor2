# 작업 실행 순서

## 현재 기준

| 항목 | 기준 |
|---|---|
| 인증 | Clerk |
| 데이터베이스 | Supabase |
| 파일 저장 | Supabase Storage |
| 실시간 알림 | Supabase Realtime |
| 결제 | 카카오 링크 결제 + 카카오뱅크 무통장 입금 |
| 메일 | Resend |
| 배포 | Vercel |
| 제외 | Stripe, Twilio |

## 작업 원칙

| 원칙 | 내용 |
|---|---|
| 세션 단위 | 한 세션에는 한 가지 큰 목표만 처리 |
| 완료 기준 | 화면 동작, 저장 처리, 오류 확인 중 최소 1개 이상 확인 |
| 환경변수 | 기존 값은 덮어쓰지 않고, 빈 값만 요청 후 입력 |
| 커밋 | 사용자 확인 후 진행 |
| 결제 API | 카카오 결제 링크가 없으면 임시 결제 모드로 진행 |
| 문서 갱신 | 실제 구현 기준이 바뀌면 계획 문서도 함께 수정 |
| 자동 검증 | 작업 결과를 제시한 뒤 자동 검증을 반드시 수행 (lint/build/실행 확인) |

## 1차 작업

목표: 로그인부터 상품 구매 확정, 구매 내역 확인까지 완성

### 현재 구현 상태

| 항목 | 상태 |
|---|---|
| 기본 앱 구조 | 완료 |
| Clerk 로그인 연결 | 완료 |
| 디자인 기준 재작업 | 완료 |
| 푸터/정책 링크 | 완료 |
| Supabase 연결 코드 | 완료 |
| 사용자 기본 정보 저장 코드 | 완료 |
| 아이디 등록 코드 | 완료 |
| 상품 목록/상세 | 완료 |
| 결제 방식 선택 | 완료 |
| 임시 결제 완료 처리 | 완료 |
| 무통장 입금 대기 주문 | 완료 |
| 주문 확정 저장 | 완료 |
| 구매 완료 메일 | Resend 키가 있으면 발송, 없으면 건너뜀 |
| 마이페이지 구매 내역 | 완료 |
| 남은 외부 준비 | Supabase 테이블 적용, Resend 키 입력, 카카오 링크/입금 계좌 입력 |

| 순서 | 작업 | 필요한 값 | 완료 기준 |
|---|---|---|---|
| 1 | 프로젝트 기본 뼈대 생성 | 없음 | 개발 서버 실행 가능 |
| 2 | Clerk 로그인 연결 | Clerk 공개키, Clerk 비밀키 | 로그인/로그아웃 가능 |
| 3 | Supabase 연결 | Supabase 주소, 공개키, 서비스 키 | DB 연결 확인 |
| 4 | 사용자 기본 정보 저장 | Clerk + Supabase | 첫 로그인 사용자 정보 저장 |
| 5 | 아이디 등록 | Supabase | 중복 확인 후 아이디 저장 |
| 6 | 상품 목록/상세 | Supabase | 상품 조회 가능 |
| 7 | 결제 방식 선택 | 상점명, 계좌 정보 | 카카오 링크/무통장 선택 가능 |
| 8 | 임시 결제 모드 | 없음 | 개발 중 결제 완료 처리 가능 |
| 9 | 무통장 입금 대기 | 계좌 정보 | 입금 대기 주문 저장 |
| 10 | 주문 확정 | Supabase | 구매 확정 상태 저장 |
| 11 | 구매 완료 메일 | Resend 키 | 주문 확정 시 메일 발송 |
| 12 | 마이페이지 구매 내역 | Supabase | 구매 내역과 다운로드 접근 가능 |
| 13 | 1차 흐름 테스트 | 전체 1차 작업 | 로그인 → 상품 → 결제 → 구매 내역 확인 |

## 2차 작업

목표: 구매 권한, 커뮤니티, 운영 기능 확장

| 순서 | 작업 | 필요한 값 | 완료 기준 |
|---|---|---|---|
| 1 | 게시판 목록/상세 | Supabase | 글 목록과 상세 확인 |
| 2 | 게시글 작성/수정/삭제 | Clerk + Supabase | 로그인 회원 글 작성 가능 |
| 3 | 댓글/대댓글 | Supabase | 댓글 작성과 삭제 가능 |
| 4 | 좋아요 | Supabase | 좋아요 추가/취소 가능 |
| 5 | 구매회원 전용 접근 | 주문 확정 데이터 | 구매자만 전용 게시판 접근 |
| 6 | 알림 | Supabase Realtime | 댓글/좋아요 알림 생성 |
| 7 | 관리자 상품 관리 | 관리자 권한 | 상품 등록/수정/삭제 가능 |
| 8 | 관리자 주문 관리 | 주문 데이터 | 입금 확인/주문 상태 변경 가능 |
| 9 | 카카오 실제 결제 링크 연결 | 카카오 결제 링크 | 결제 버튼 실제 링크 이동 |
| 10 | 카카오 로그인 추가 | Clerk 카카오 설정 | 카카오 로그인 가능 |
| 11 | Google Analytics | 측정 ID | 방문 통계 연결 |
| 12 | 2차 통합 테스트 | 전체 기능 | 권한/알림/관리자 흐름 확인 |
